# TechSave - Modern E-Commerce Platform 🛒💻

TechSave ek fully functional e-commerce web application hai jo specifically electronics aur tech gadgets ke liye design ki gayi hai. Is project ko **React.js** aur modern web technologies ka use karte hue develop kiya gaya hai aur ye **Vercel** par live deploy hai.

## ✨ Key Features

*   **Responsive Design:** Mobile-first approach ke sath banayi gayi hai taake har screen size (Mobile, Tablet, Desktop) par perfect dikhayi de.
*   **Product Management:** Products ko browse karna, categories filter karna, aur detail view dekhna.
*   **Shopping Cart:** Items ko cart mein add karna, quantity update karna, aur remove karne ki mukammal saholat.
*   **State Management:** Smooth user experience ke liye efficiently data handle kiya gaya hai.
*   **Modern UI/UX:** Clean, dark/light theme (agar apply ki hai) aur tech-focused interface.

---

## 🛠️ Tech Stack

*   **Frontend:** React.js
*   **Styling:** Tailwind CSS (Responsive & Modern)
*   **Deployment:** Vercel
*   **Icons:** Lucide-React / FontAwesome
